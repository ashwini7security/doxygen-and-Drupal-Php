DatabaseStatementBase::fetchField ( $  index = 0)

Sets the default fetch mode for this statement.

See http://php.net/manual/en/pdo.constants.php for the definition of the constants used.

Parameters:
$modeOne of the PDO::FETCH_* constants.
$a1An option depending of the fetch mode specified by $mode:
  • for PDO::FETCH_COLUMN, the index of the column to fetch
  • for PDO::FETCH_CLASS, the name of the class to create
  • for PDO::FETCH_INTO, the object to add the data to
$a2If $mode is PDO::FETCH_CLASS, the optional arguments to pass to the constructor. Fetches the next row from a result set.

See http://php.net/manual/en/pdo.constants.php for the definition of the constants used.

Parameters:
$modeOne of the PDO::FETCH_* constants. Default to what was specified by setFetchMode().
$cursor_orientationNot implemented in all database drivers, don't use.
$cursor_offsetNot implemented in all database drivers, don't use.
Returns:
A result, formatted according to $mode. Returns a single field from the next record of a result set.
Parameters:
$indexThe numeric index of the field to return. Defaults to the first field.
Returns:
A single field from the next record, or FALSE if there is no next record.
