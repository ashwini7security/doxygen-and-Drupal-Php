Format a table cell.

Adds a class attribute to all cells in the currently active column.

Parameters:
$cellThe cell to format.
$headerAn array of column headers in the format described in theme_table().
$tsThe current table sort context as returned from tablesort_init().
$iThe index of the cell's table column.
Returns:
A properly formatted cell, ready for _theme_table_cell().
