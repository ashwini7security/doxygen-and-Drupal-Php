API function to add a language.

Parameters:
$langcodeLanguage code.
$nameEnglish name of the language
$nativeNative name of the language
$directionLANGUAGE_LTR or LANGUAGE_RTL
$domainOptional custom domain name with protocol, without trailing slash (eg. http://de.example.com).
$prefixOptional path prefix for the language. Defaults to the language code if omitted.
$enabledOptionally TRUE to enable the language when created or FALSE to disable.
$defaultOptionally set this language to be the default.
